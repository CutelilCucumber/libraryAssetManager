# VTT Asset Library Manager

A full-stack inventory-style application designed for managing virtual tabletop (VTT) assets such as tokens, map tiles, and icons. This project fulfills CRUD requirements while serving as a foundational system for a larger VTT platform.

---

## Overview

The VTT Asset Library Manager allows users to organize and manage collections of visual assets used in tabletop-style applications. It provides structured storage, categorization, and editing of assets that can later be integrated into a real-time multiplayer map system.

This project is built as part of a full-stack learning curriculum and emphasizes:
- RESTful API design
- Relational database modeling
- Server-side rendering and routing
- Clean project structure
